# UvA Courses

A web app with which users can shop for University of Amsterdam courses.

Features:

* Sign in 

* Change password

* Search for courses

* Select courses

* Overview of selected courses

* Delete selected courses

* Sign out

Frameworks / languages / libraries :

* Bootstrap 

* HTML5

* Javascript

* Rails 4.0.2

* Ruby 1.9.3

* (S)CSS

* SQLite
